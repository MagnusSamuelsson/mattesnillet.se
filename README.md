# 🧠 Mattesnillet

**Mattesnillet** är ett svenskt mattespel där barn, unga och vuxna kan träna huvudräkning på ett roligt, snabbt och lättillgängligt sätt. Spelet fungerar direkt i webbläsaren – inget konto krävs.

Perfekt för skolor, lärare, elever, föräldrar eller dig som bara vill hålla hjärnan i trim.

---

## 🚀 Funktioner

- Spelaren anpassar efter sin nivå
- För träning på multiplikation och division
- Snyggt, snabbt och mobilvänligt gränssnitt
- Inga inloggningar, inga annonser, inget krångel
- Fungerar offline (PWA)
